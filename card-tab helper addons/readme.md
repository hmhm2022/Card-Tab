# Card-Tab Helper 浏览器扩展

管理 Card-Tab 书签的浏览器扩展，支持当前页面书签管理、批量导入导出。

## 安装

1. 下载扩展 [card-tab-helper.zip](https://github.com/hmhm2022/Card-Tab/releases) 并解压
2. Chrome 浏览器访问 `chrome://extensions/`，开启「开发者模式」
3. 点击「加载已解压的扩展程序」，选择解压后的 `card-tab helper` 文件夹

## 功能

- **当前页面管理**：自动检测当前页面是否已收藏，支持一键添加/删除/修改分类
- **批量导入**：将 Chrome 书签导入 Card-Tab（支持确认对话框和自动备份）
- **批量导出**：备份 Card-Tab 书签为 JSON 文件（支持选择保存路径）
- **登录有效期**：15 分钟 / 1 小时 / 1 天 / 7 天 / 30 天 / 永久

## 使用

1. 点击扩展图标，首次使用需配置 API 地址并登录
2. 当前页面书签状态自动检测，可直接添加/删除/修改分类
3. 底部按钮进行批量导入导出操作
